# 耘初茶食 · FB 廣告投放邏輯（Playbook）

> 一句話核心：**廣泛投放時代，演算法負責找人、你負責餵素材。**
> 投放邏輯＝用一套分層框架，讓「策略方向 → 漏斗結構 → 定向 → 素材引擎 → 預算 → 衡量」環環相扣。
> 製作日 2026-06；依據：你 2 年真實數據（ROAS 2.46、CBO 2.83 > ABO 2.30、冬旺夏淡）＋ 2025-26 業界主流做法。

---

## 第 0 層｜策略方向：安索夫矩陣（決定「往哪長」）

安索夫**不是 FB 設定**，是最上層的方向決策。決定完，才往下變成 campaign。

| 象限 | 對耘初 | 風險 | 配置 |
|---|---|---|---|
| **市場滲透**（舊產品×舊市場） | 把更多茶賣給同一批台灣養生客 | 最低 | **主力** |
| 產品開發（新產品×舊市場） | 新組合/新品賣給現有客 | 中 | 組合包＋測試 |
| 市場開發（舊產品×新市場） | 同樣的茶賣給新受眾（送禮/不同年齡/海外） | 中 | 測試探索 |
| 多角化（新產品×新市場） | 風險最高 | 高 | ❌ 不投廣告 |

**鐵則：90% 滲透 + 10% 探索。** 你產品已驗證、台灣還有空間 → 重心就在滲透。安索夫的價值是「提醒你別把 10% 探索預算也丟進滲透，要拿去試相鄰象限」。

## 第 1 層｜漏斗結構：TOFU / MOFU / BOFU（決定「怎麼接觸」）

| 階段 | 你的活動 | 受眾 | 預算 |
|---|---|---|---|
| **TOFU 冷** | 主力 全受眾 | 廣泛 Advantage+ | $2,100 |
| **MOFU/BOFU 暖熱** | 再行銷 | 訪客/加購未購/舊客 | $600 |
| **測試** | 測試組 | 廣泛（探索） | $300 |

> ⚠️ 現代修正：傳統說「TOFU 用觸及/awareness」。但**低價消費品＋Advantage+**，TOFU 應該**直接用購買最佳化＋廣泛**讓 Meta 自己找買家——你的主力就是這樣設，是對的。
> ⚠️ 消費品例外：一般說「購買者要排除」，但**茶是會回購的消耗品**，所以再行銷保留「舊客回購」區隔是合理的（一般電商不會這樣做）。

## 第 2 層｜定向哲學：廣泛 + Advantage+，「素材就是定向」

業界 2025-26 共識，也被你的數據印證：
- 廣泛/Advantage+ 比類似受眾(LLA) **ROAS 高 ~49%**；Advantage+ 創意功能 **+22% ROAS**
- **你的數據**：CBO 廣泛 ROAS **2.83** > ABO/興趣 **2.30** → 同一個結論
- 做法：**不要開 5-10 個興趣廣告組；開 2-3 個廣泛廣告組，用「素材角度」區隔，不是用受眾區隔**
- 績效差異幾乎全來自素材，不是受眾

→ **結論：受眾保持廣泛、別手動細分。你能控制、且決定成敗的，是素材。**

## 第 3 層｜素材引擎：三階段測試（這是核心引擎）

| 階段 | 問題 | 怎麼做 |
|---|---|---|
| ① Pre-flight | 哪支新素材最好？ | 在**測試組**跑，新素材**互相比**（別跟舊冠軍比） |
| ② New vs BAU | 新素材有沒有比現有冠軍好？ | 勝出者放進主力跟現役同跑比較 |
| ③ Scale | 怎麼放大贏家？ | 升級進主力，預算漸進加 |

**規則**：
- 一支素材被同一人看 **6 次以上**，購買意願掉 ~16% → **疲勞**
- 頂尖品牌平均 **每 10 天**換一批新素材
- 加新素材**不要急著關舊的**，讓它們並跑
- **影片 > 圖片**（你的數據：影片 2.64 > 圖片 2.46）；**UGC/素人感 > 精緻廣告**

> 📌 你現在的測試組放的是「被砍太早的低觸及舊素材」——這是**一次性的素材回收挖礦**，挖完之後，測試組的長期任務要回到「**測真正的新素材**」（新 UGC、新角度、組合促銷頁專屬）。

---

## 第 4 層｜預算邏輯：70/20/10 + 放大紀律

- **分配**：主力 70%（$2,100）｜再行銷 20%（$600）｜測試 10%（$300）= $3,000/日
- **學習期**：每個廣告組需 **~50 購買/週** 才穩定；不足就別細分
- **放大**：贏家一次加 **+20~30%**，間隔 **48-72 小時**，避免觸發重新學習
- **止損（3x Kill）**：單組花到 3× 目標 CPA（~$1,400）仍 0 購買 → 關
- **別頻繁改**：上線後 3-5 天、累積 ~50 購買前不要動

## 第 5 層｜衡量與節奏

- **追蹤**：GA4 + FB Pixel/CAPI（已驗證在跑）；唯購買事件待一筆真實訂單最終確認
- **ROAS 目標（依季節）**：
  - 旺季 11-2 月：衝 **3.0-3.3**、加碼預算（你最佳月 3.36）
  - 淡季 6-9 月：守 **2.4-2.6**，靠「冷泡冰涼」角度對抗掉量
- **客單價槓桿**：組合包 $1,050 拉 AOV → 直接墊高 ROAS（ROAS = AOV ÷ CPA）
- **每週節奏**：`npm run pull` 拉資料 → `/投放` 檢視 → 關賠錢組、好組 +20%、補新素材

## 每週決策樹（貼牆版 · 照著跑）

> 🟢 綠＝擴大順勢　🟠 琥珀＝修正止損　⚪ 灰＝判斷/維持

```
每週開始 ─► 拉資料 (npm run pull) ─► 跑 /投放 檢視
                       │
   ①  主力 ROAS 達標？
        ├─ 是 ─► 🟢 好素材 +20% 預算
        └─ 否 ─► 🟠 暫停拖累素材 → 補新素材
                       │
   ②  測試素材贏過主力現役？
        ├─ 有 ─► 🟢 升級進主力（舊的並跑）
        └─ 無 ─► ⚪ 換下一批新素材再測
                       │
   ③  再行銷頻率過高（疲勞）？
        ├─ 是 ─► 🟠 控頻 / 換素材
        └─ 否 ─► ⚪ 維持現狀
                       │
   ④  現在旺季(11–2月) 還是 淡季(6–9月)？
        ├─ 旺 ─► 🟢 加碼預算，衝 ROAS 3.0–3.3
        └─ 淡 ─► 🟢 守 2.4–2.6，主打冷泡冰涼
                       │
   每月 ─► 回看安索夫：滲透還有空間？拿 10% 測一個新象限
```

**不變的框（每週都成立）**
- 受眾廣泛不細分 · 50 購買/週才穩 · 放大每次 +20%（間隔 48–72h）· 每 10 天換素材 · 影片 > 圖片
- 預算 70/20/10：主力 $2,100 · 再行銷 $600 · 測試 $300

## 這套邏輯的層次關係（為什麼這樣排）

- **安索夫** = 管「長什麼方向」（季度策略）
- **漏斗** = 管「怎麼接觸」（活動結構）
- **定向哲學** = 管「受眾交給誰決定」（→ 交給演算法，廣泛）
- **素材引擎** = 管「你實際能控制的變因」（→ 持續產新素材測試）
- **預算/紀律** = 管「資源怎麼分、何時放大何時止損」
- **衡量** = 管「怎麼知道有沒有效、何時調整」

> 越上層越少變、越下層越常動。日常 90% 的工夫花在**素材引擎**那一層。
